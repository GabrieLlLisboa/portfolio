As linguagens!

As linguagens tem três tipos, a de Marcação, Estilização e as de Programação. Mas quais são elas?

HTML

HTML é uma linguagem de Marcação. usada para estruturar páginas da web. Ela organiza textos, títulos, imagens, links, listas e outras partes do conteúdo para que o navegador mostre corretamente. Não é uma linguagem de programação, e sim de marcação. Sozinha, define apenas a estrutura; o CSS dá a aparência e o JavaScript traz a interatividade.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas web, definindo cores, fontes, tamanhos, espaçamentos, layout e animações de elementos. Criado em 1996, permite separar o conteúdo da apresentação. Pode ser aplicado de forma direta nos elementos, internamente no documento ou externamente em arquivos separados. Funciona através de seletores que apontam elementos e propriedades que definem como eles aparecem. CSS também permite criar responsividade, animações, efeitos visuais e controlar o posicionamento de elementos na página.

JavaScript

JavaScript é uma linguagem de programação usada para tornar páginas web interativas e dinâmicas. Permite manipular elementos da página, responder a ações do usuário, validar formulários, criar animações, atualizar conteúdos sem recarregar a página e se comunicar com servidores. Criada em 1995, funciona em navegadores e também no servidor com ambientes como Node.js. JavaScript usa variáveis, funções, objetos e eventos para controlar o comportamento dos elementos, sendo essencial no desenvolvimento front-end junto com HTML e CSS.

C#

C# é uma linguagem de programação moderna, criada pela Microsoft em 2000, usada principalmente no desenvolvimento de aplicações Windows, web, jogos com Unity e softwares corporativos. É orientada a objetos, fortemente tipada e permite escrever código seguro e estruturado. Suporta conceitos como classes, herança, interfaces, eventos e tratamento de exceções. C# roda na plataforma .NET, que fornece bibliotecas e ferramentas para criar aplicações de diferentes tipos, incluindo desktop, web, mobile e serviços de backend.
