As linguagens!

As linguagens tem três tipos, a de Marcação, Estilização e as de Programação. Mas quais são elas?

HTML

HTML é a linguagem usada para estruturar páginas da web. Ela organiza textos, títulos, imagens, links, listas e outras partes do conteúdo para que o navegador mostre corretamente. Não é uma linguagem de programação, e sim de marcação. Sozinha, define apenas a estrutura; o CSS dá a aparência e o JavaScript traz a interatividade.
